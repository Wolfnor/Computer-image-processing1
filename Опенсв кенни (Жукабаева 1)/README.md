# Простой проект распознавания объектов на OpenCV

Простой проект для распознавания объектов через веб-камеру с использованием OpenCV и предобученной модели MobileNet-SSD.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Скачайте файлы модели MobileNet-SSD автоматически:
```bash
python download_model.py
```

Или скачайте вручную:
   - `MobileNetSSD_deploy.prototxt` - описание архитектуры модели
   - `MobileNetSSD_deploy.caffemodel` - веса модели (~23 МБ)
   
   Их можно скачать здесь: https://github.com/chuanqi305/MobileNet-SSD

## Запуск

```bash
python object_detection.py
```

Нажмите 'q' для выхода из программы.

## Что делает программа

1. Открывает веб-камеру
2. В реальном времени распознает объекты на видео
3. Рисует рамки вокруг найденных объектов
4. Показывает название объекта и процент уверенности
